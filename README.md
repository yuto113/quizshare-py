# QuizShare（クイズシェア）

グループのみんなでクイズを出し合って、解き合って、競い合えるWebサービスです。
合言葉(グループID)でログインするだけで、登録なしですぐ使えます。

🔗 **本番サイト**: https://yuto113.pythonanywhere.com

## 主な機能

- 📝 クイズの投稿(選択式・記述式、画像添付、ヒント、解説、別解にも対応)
- ⏱️ タイマー計測つきの解答と、正答率・タイムのランキング
- ⚔️ バトルモード(ルームコードで友達とリアルタイム対戦)
- 🎉 イベント機能(期間限定の大会、結果発表日時の予約)
- 🏷️ クイズセット、タグ、テーマ別ページ、ランダム出題
- 👑 管理者ページ(クイズの管理、利用規約違反の調査中フラグ、統計)
- 🏢 スタッフ専用エリア(社内チャット、ファイル共有+プレビュー、タスク、カレンダー、人事管理)

## 技術構成

- **バックエンド**: Python / Flask
- **データベース**: SQLite
- **フロントエンド**: HTML / Jinja2 / JavaScript(部分更新でサクサク動作)
- **ホスティング**: PythonAnywhere

## セキュリティへの取り組み

- 問題文・答え・作者名などのコンテンツは **Fernet方式で暗号化**して保存
- グループIDは **HMAC-SHA256**、パスワードは **scrypt** でハッシュ化
- 秘密鍵・パスワードはすべて環境変数で管理(コードには含めない)
- 採点や時間の判定は**すべてサーバー側**で実施(ブラウザの自己申告を信用しない設計)
- イベントの不正参加対策(参加券方式)、レートリミット、エラー監視
- 毎日の自動バックアップ(DBは7日分、画像は永久アーカイブ)

## 運営ポリシー

利用規約とプライバシーポリシーを整備し、著作権に配慮した運営をしています。
利用規約違反の疑いがあるコンテンツは「調査中」として公開を一時停止する仕組みがあります。
